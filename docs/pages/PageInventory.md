# Pages 인벤토리

| Page | Template | 주요 Organisms |
|------|----------|----------------|
| 홈 | HomeTemplate | Hero, TrustBadges, ProductGrid, ColorPaletteSection |
| 제품 목록 | ProductListTemplate | SearchBar, ProductGrid |
| 제품 상세 | ProductDetailTemplate | ColorPreview, ProductGrid |
