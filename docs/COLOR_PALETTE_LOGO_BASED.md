# 카리페인트 로고 기반 색상 팔레트

> 로고의 시각적 정체성을 반영한 색상 조합 가이드

---

## 1. 로고 색상 분석

### 1.1 로고 구성 요소
- **텍스트**: "Kari Paint" – 세리프 폰트, **블랙**
- **아이콘**: 왕관 + 페인트 도구 – **딥 마룬/버건디 레드**
- **배경**: 원형 프레임 내 **따뜻한 오프화이트/크림**

### 1.2 로고가 전달하는 메시지
- **마룬**: 품질, 열정, 전문성, 프리미엄, 장인정신(왕관 → 위엄)
- **블랙**: 신뢰, 확고함, 세련됨
- **크림**: 따뜻함, 환영, 클린, 정돈된 공간감

---

## 2. 기존 팔레트의 문제점

| 기존 | 로고 | 충돌 |
|------|------|------|
| Primary #1B5E7A (블루) | 마룬 레드 | 로고와 무관한 색상 |
| Secondary #E8A735 (골드) | — | 마룬과 잘 맞지 않는 선명한 오렌지 |
| 배경 #FAFAFA (냉청회) | 크림 | 차가운 느낌 |

---

## 3. 새 색상 전략

### 3.1 핵심 원칙
1. **로고의 마룬**을 Primary로 사용
2. **블랙**은 본문·제목 기본 텍스트로 유지
3. **따뜻한 크림/오프화이트**를 배경 톤으로 사용
4. **Secondary**는 마룬과 조화되는 골드/참파뉴 계열

### 3.2 마룬 & 골드 조합의 의미
- 마룬 + 골드: 왕관·품질·전문성과 잘 어울리는 조합
- 참파뉴/골드는 CTA, 포인트, 강조에 적합
- 과한 대비보다는 차분한 톤 유지

---

## 4. 제안 색상 팔레트

### 4.1 Primary (로고 마룬)
```yaml
primary:
  main: "#722F37"    # 로고 아이콘과 동일/근접한 딥 마룬
  light: "#8B3D47"   # 호버, 배지, 연한 배경
  dark: "#5A242B"    # 강조, 액티브, 다크 모드
```
- **#722F37**: 버건디에 가까운 마룬, 로고와 조화
- **#8B3D47**: 밝은 마룬, 배지·라벨 등에 사용
- **#5A242B**: 어두운 마룬, 버튼·링크 강조 등

### 4.2 Secondary (참파뉴 골드)
```yaml
secondary:
  main: "#B8860B"    # 다크 골드 – 로고와 균형
  light: "#D4AF37"   # 골드 – CTA, 포인트
  dark: "#8B6914"    # 다크 골드
```
- 기존 선명한 오렌지 대신 **골드/참파뉴**로 교체
- 왕관·프리미엄 이미지와 맞음

### 4.3 Neutrals (따뜻한 톤)
```yaml
neutral:
  background: "#FAF8F5"   # 크림 배경 (로고 배경 톤)
  background-alt: "#F5F3EE"
  gray-50: "#FAFAF8"
  gray-100: "#F0EDE8"     # 따뜻한 라이트 그레이
  gray-200: "#E5E0D9"
  gray-300: "#D4CFC6"
  gray-400: "#A39E95"
  gray-500: "#6B6359"     # 따뜻한 미드 그레이
  gray-600: "#554D44"
  gray-700: "#3D3630"
  gray-800: "#2A2520"
  gray-900: "#1A1A1A"     # 블랙에 가까운 텍스트
  black: "#000000"
```
- 냉청 그레이 대신 **따뜻한 그레이** 사용
- 배경은 로고의 크림 느낌을 살림

### 4.4 Semantic (유지·미세 조정)
```yaml
semantic:
  success: "#2E7D32"   # 그린 – 그대로
  warning: "#B8860B"   # secondary와 통일
  error: "#8B2635"     # 마룬 계열 에러
  info: "#5A242B"      # primary dark
```

## 5. 색상 조합 예시

| 용도 | 조합 | 비고 |
|------|------|------|
| CTA 버튼 | Primary main on Background | 강한 대비 |
| 보조 CTA | Secondary main on Background | 포인트 |
| 카드 배경 | #FFFFFF on Background | 카드에 화이트 사용 |
| 본문 텍스트 | gray-900 on Background | 가독성 확보 |
| 부가 텍스트 | gray-600 on Background | 구분감 |

---

## 6. 피해야 할 조합
- Primary + 선명한 오렌지/레드 (시선 산만)
- 배경에 냉청 그레이 (#F5F5F5 등) 과다 사용
- Pure White (#FFFFFF)만의 배경 – 크림과 섞어 사용
