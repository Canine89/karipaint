# Organism: InquiryBanner

## 개요
블로그·홈페이지 글마다 하단에 노출하는 **견적 문의 배너**.  
"편한 방법으로 부담 없이 문의 주세요" 메시지와 다중 채널 CTA.

## 구성
- Label: "📩 견적 문의 방법 안내" 또는 "편하신 방법으로 언제든지 연락 주세요"
- ContactCta[]: 카카오톡, 인스타그램, 네이버 톡톡
- Optional: 안내 문구 "빠르고 친절하게 상담 도와드리겠습니다"

## 레이아웃 예시
```
┌─────────────────────────────────────────────┐
│ 📩 견적 문의 방법 안내                        │
│ 편하신 방법으로 언제든지 연락 주세요 😊        │
│                                             │
│ 🟡 1:1 카카오톡  📸 인스타 DM  💚 네이버 톡톡   │
│    [카카오톡 문의]  [인스타 DM]  [톡톡 문의]   │
│                                             │
│ 가장 편한 방법으로 부담 없이 문의 주세요.      │
│ 빠르고 친절하게 상담 도와드리겠습니다.        │
└─────────────────────────────────────────────┘
```

## Props
- `channels`: `{ kakao?: string; instagram?: string; naverTalk?: boolean }`
- `compact?: boolean` – 간소화 버전 (아이콘만 또는 한 줄)
