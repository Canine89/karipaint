# Organism: Hero

## 구성
- 배경 (이미지 / 그라데이션 / 단색)
- Label (메인 헤드라인)
- Label (서브 카피)
- Button[] (CTA 1~2개)

## Variants
- `full` – 전체 너비, 큰 타입
- `centered` – 중앙 정렬
- `split` – 좌측 텍스트, 우측 이미지

## 카리페인트 맞춤 (BRAND_STORY 기반)
- **메인 메시지 예시**: "공간의 가치를 높이는, 내 손으로 완성하는"
- **서브 카피 예시**: "2대째 이어온 도장 시공, 성취감과 감동을 전합니다"
- 비주얼: 현장감·손길이 느껴지는 이미지, 공간 Before/After 느낌
