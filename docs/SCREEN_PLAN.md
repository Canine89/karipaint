# 카리페인트 웹사이트 화면 계획

> 3개 메뉴 기준 화면 구성 및 디자인 방향

---

## 1. 메뉴 구조

| 메뉴 | 경로 | 용도 |
|------|------|------|
| **랜딩 페이지** | `/` | 첫인상, 브랜드 소개, CTA 유도 |
| **견적 문의** | `/inquiry` | 1:1 카톡 연결 + 견적 문의 방법 안내 |
| **작업 결과물** | `/portfolio` | 시공 사례/갤러리 |

---

## 2. 공통 요소

### Header (모든 페이지)
- 로고
- 네비: 랜딩 | 견적 문의 | 작업 결과물
- 카톡/문의 CTA 버튼 (우측)

### Footer (모든 페이지)
- 로고, 사업자 정보
- 링크: 견적 문의, 작업 결과물
- 문의 채널 (카톡, 인스타, 톡톡)

---

## 3. 화면별 상세 계획

### 3.1 랜딩 페이지 (`/`)

**목적**: 첫 방문자에게 브랜드 전달, 작업 결과물·견적 문의로 유도

**구성**:
```
[Header]
[Hero]
  - 메인 메시지: "공간의 가치를 높이는, 내 손으로 완성하는"
  - 서브: "2대째 이어진 도장 시공"
  - CTA: 견적 문의하기
[TrustBadges]
  - 2대 계승, 기술 기반, 고객 만족
[작업 결과물 미리보기]
  - 최근 3~4개 카드 + 전체보기 링크
[회사 소개 요약]
  - 한 문장 + 성취감·감동·성장 3가지
[InquiryBanner]
  - 편한 방법으로 문의하세요 + 카톡/인스타/톡톡
[Footer]
```

---

### 3.2 견적 문의 (`/inquiry`)

**목적**: 1:1 카톡 즉시 연결 + 견적 문의 절차 안내

**구성**:
```
[Header]
[Hero - 간소화]
  - "견적 문의"
  - "편한 방법으로 언제든 연락 주세요"
[1:1 카톡 연결 영역]
  - 큰 카카오톡 CTA 버튼
  - "실시간으로 빠르게 상담드립니다"
[견적 문의하는 법]
  - 단계별 안내 (1. 연락 2. 현장 확인 3. 견적 제출 등)
[다른 채널]
  - 인스타 DM, 네이버 톡톡
[Footer]
```

---

### 3.3 작업 결과물 (`/portfolio`)

**목적**: 시공 사례 갤러리로 신뢰·품질 전달

**구성**:
```
[Header]
[Hero - 간소화]
  - "작업 결과물"
  - "공간을 변화시킨 시공 사례"
[필터/분류]
  - 인테리어 도장 | 외벽 도장 | 기타 (선택)
[갤러리 그리드]
  - 카드: 썸네일, 제목, 간단 설명
  - Before/After 옵션
[Footer]
```

---

## 4. 디자인 시퀀스

1. **랜딩 페이지** – 전체 톤·구조 확정
2. **견적 문의** – CTA·문의 흐름 설계
3. **작업 결과물** – 갤러리 레이아웃·카드 패턴 확정

---

## 5. 반응형 (PC/모바일)

| 화면 | 모바일 (375px) | PC (1024px) |
|------|----------------|-------------|
| **랜딩** | 세로 스택, Hero → Trust → Portfolio → Inquiry | Hero 가로(텍스트+이미지), Trust 가로, Portfolio 3열 그리드, Inquiry 버튼 가로 |
| **견적 문의** | 카톡 CTA → 단계 안내 → 다른 채널 | 2열(카톡 영역 \| 견적 방법+다른채널) |
| **작업 결과물** | 필터 → 카드 세로 스택 | 필터 → 3열 갤러리 그리드 |

**Pencil**: 모바일(위) + PC(아래) 나란히 배치
