# Molecule: ContactCta

## 구성 Atoms
- Icon (phone / mail / 카톡 / 인스타 / 톡톡)
- Label
- Link 또는 Button

## 용도
- 헤더, 푸터의 연락처 노출
- CTA 섹션
- 견적 문의 배너

## 카리페인트 문의 채널 (BRAND_STORY 참조)
| 채널 | 아이콘 | 링크 placeholder |
|------|--------|-------------------|
| 카카오톡 1:1 | 🟡 | `https://open.kakao.com/o/...` |
| 인스타그램 DM | 📸 | `https://instagram.com/...` |
| 네이버 톡톡 | 💚 | 블로그 상단/하단 톡톡 버튼 |

## Variants
- `inline` – 한 줄에 아이콘+텍스트
- `stacked` – 아이콘 위, 텍스트 아래
- `banner` – "편하신 방법으로 언제든 문의하세요" + 여러 채널 나열
