# 인테리어 업체 느낌 디자인 점검 피드백

> 웹 검색 기반 트렌드 + 현재 Pencil 디자인 분석

---

## 1. 웹 검색 기반 인테리어 업체 웹사이트 트렌드

### 1.1 컬러 & 분위기
- **따뜻하고 아늑한 톤**: 베이지, 그레이, 아이보리 계열 선호
- **2~3가지 주색**: 세련되고 모던하게 일관 사용
- **브랜드 일관성**: 로고·색상·폰트를 모든 페이지에 통일

### 1.2 콘텐츠 & 시각
- **고해상도 이미지**: 포트폴리오의 품질 = 브랜드 전문성
- **Before/After**: 시공 전후 비교 필수
- **다양한 촬영**: 와이드샷, 디테일샷, 라이프샷
- **영상**: 360도 투어, 타임랩스, 고객 인터뷰 등

### 1.3 UX & 구조
- **3클릭 내 도달**: 원하는 정보까지 직관적 네비게이션
- **모바일 최적화**: 반응형 필수, 이미지/영상도 모바일 최적화
- **포트폴리오 분류**: 주거/상업/사무 등 카테고리 제공

### 1.4 신뢰 구축
- **실제 고객 리뷰**: 리뷰·평가 게시
- **인증·수상 이력**: 공식 인증서, 협력사 로고
- **블로그**: 인테리어 팁, 시공·자재 정보로 전문성 강화

## 2. 현재 디자인 점검 결과

### 2.1 잘된 점

| 항목 | 현재 상태 | 트렌드 부합 |
|------|-----------|-------------|
| **컬러 톤** | 마룬(#722F37) + 크림(#FAF8F5) + 골드(#B8860B) | 따뜻한 크림/아이보리 배경 ✓ |
| **브랜드 일관성** | 로고 기반 색상 팔레트 문서화 | 일관된 사용 가능 ✓ |
| **섹션 구성** | Hero → Trust → Portfolio → Intro → Inquiry | 직관적 흐름 ✓ |
| **TrustBadges** | 2대 계승, 기술, 고객 만족 | 전문성·신뢰 메시지 ✓ |
| **문의 채널** | 카톡/인스타/톡톡 | 편한 접근성 ✓ |
| **모바일·PC** | 375px / 1024px 화면 모두 구성 | 반응형 고려 ✓ |

### 2.2 개선 필요 영역

#### A. 시각적 콘텐츠 (우선순위 높음)

| 이슈 | 현재 | 권장 |
|------|------|------|
| **포트폴리오 플레이스홀더** | #E5E0D9 회색 블록 + "시공 사례 1, 2, 3" | 실제 시공 사진 또는 품질 높은 인테리어 참고 이미지 |
| **PC Hero 이미지** | 0PNHu – 단색(#8B3D47) 프레임 | 시공 현장/완료 공간 사진 |
| **Before/After 부재** | 없음 | 시공 전후 비교 섹션 또는 카드 옵션 추가 |

→ 인테리어 업체 이미지는 **사진 품질**에 크게 좌우됨. 플레이스홀더를 실제 느낌에 가까운 이미지로 교체하는 것이 가장 중요.

#### B. 색상 & 톤

| 이슈 | 현재 | 권장 |
|------|------|------|
| **인스타/톡톡 버튼** | #E4405F(핑크), #03C75A(녹색) 등 채도 높은 색 | Primary/Secondary 범위 내로 낮춘 톤, 또는 아웃라인 스타일 |
| **마룬 비중** | Hero, InquiryBanner, CTA 등 전반적 사용 | 너무 강하지 않게, 크림·화이트 비중을 늘려 여유감 강화 |

→ 전반적으로는 "따뜻하고 차분한" 인테리어 업체 느낌에 부합하나, 소셜 버튼 색이 과하다.

#### C. 포트폴리오 섹션

| 이슈 | 현재 | 권장 |
|------|------|------|
| **카테고리** | "전체/인테리어/외벽" 정도만 존재 | "거실·침실", "상업 공간", "외벽" 등 세분화 검토 |
| **카드 정보** | 제목·간단 설명 위주 | 사용 자재, 시공 기간, 고객 후기 일부 노출 고려 |
| **Before/After** | 없음 | 필터 또는 카드 타입으로 Before/After 추가 |

#### D. 신뢰·사회적 증거

| 이슈 | 현재 | 권장 |
|------|------|------|
| **고객 후기** | 없음 | 랜딩 또는 전용 섹션에 리뷰/평가 2~3개 |
| **인증·협력** | 없음 | 공사업 등록, 협력사 로고 등 (있다면) |
| **시공 수·경력** | TrustBadges에만 간략 | "OO건 시공", "OO년 경력" 등 숫자로 구체화 |

#### E. 타이포·여백

| 이슈 | 현재 | 권장 |
|------|------|------|
| **히어로 메시지** | "공간의 가치를 높이는, 내 손으로 완성하는" | 인테리어 업체에 맞게 유지 가능, 필요 시 미세 조정 |
| **섹션 간격** | 24~40px | 넉넉한 여백으로 공간감 강조 |

→ Inter + 크림 배경 조합은 인테리어 사이트에 잘 맞는 선택.

#### F. 디테일·미적 요소

| 이슈 | 현재 | 권장 |
|------|------|------|
| **텍스처·패턴** | 없음 | 크림 배경에 은은한 텍스처 또는 패턴 검토 (과하지 않게) |
| **아이콘** | 기본형 | 마룬·골드 계열과 어울리는 커스텀 아이콘 고려 |
| **카드 그림자** | 있음 | 인테리어 스타일에 맞게 부드러운 shadow 유지 |

## 3. 우선순위별 개선 제안

### P0 (실제 구현 시 필수)
1. **포트폴리오 이미지**: 플레이스홀더 → 실제 시공 사진으로 교체
2. **PC Hero**: 배너 오른쪽 영역에 시공/완료 공간 이미지 삽입
3. **소셜 버튼 색상**: Primary/Secondary 범위로 조정

### P1 (인테리어 업체 느낌 강화)
4. Before/After 섹션 또는 카드 타입 추가
5. 고객 후기 2~3개 섹션
6. 시공 수/경력 숫자로 구체화

### P2 (추가 개선)
7. 포트폴리오 카테고리 세분화
8. 은은한 배경 텍스처/패턴
9. 인증·협력사 로고(있는 경우)

## 4. 색상 조정 예시 (Pencil 적용 가능)

```
소셜 버튼 대안:
- 인스타: Primary light(#8B3D47) 또는 아웃라인 + #6B6359
- 톡톡: Primary dark(#5A242B) 또는 Secondary dark(#8B6914)

→ 마룬·골드 팔레트 내에서 조화되게 조정
```

## 5. 요약

| 구분 | 평가 |
|------|------|
| **현재 톤** | 마룬+크림+골드로 인테리어 업체에 어울리는 기반 확보 |
| **핵심 과제** | 플레이스홀더 → 실제 시공 이미지로 전환, 소셜 버튼 색 조정 |
| **추가 강화** | Before/After, 고객 후기, 시공 수/경력 강조 |

인테리어 업체 웹사이트 트렌드와 비교했을 때, 현재 구조와 컬러 선택은 적절하고, **실제 이미지 교체와 소셜 버튼 색 정리**를 우선 적용하면 인테리어 업체 느낌이 크게 향상될 것으로 보입니다.
